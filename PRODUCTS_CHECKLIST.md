## 📝 Instrucciones de Reemplazo de Imágenes

### 1. Buscar en CJ Dropshipping
1. Ve a [cjdropshipping.com](https://cjdropshipping.com)
2. Usa el buscador con el SKU (ej: "CJDOG001")
3. Accede a la página del producto
4. Descarga todas las imágenes disponibles

### 2. Formato de Imágenes Recomendado
- **Formato:** JPG o WebP
- **Resolución principal:** 800x800px mínimo
- **Resolución galería:** 1200x1200px
- **Tamaño máximo:** 500KB por imagen
- **Fondo:** Blanco o transparente para productos

### 3. Estructura de Carpetas
```
public/products/
├── [SKU]/
│   ├── main.jpg           ← Imagen principal (obligatoria)
│   ├── gallery/
│   │   ├── 01.jpg        ← Vista adicional 1
│   │   ├── 02.jpg        ← Vista adicional 2
│   │   └── 03.jpg        ← Vista adicional 3
│   └── 3d/               ← Solo si has3D=true
│       ├── frame_0001.webp
│       └── ...
```

### 4. Actualizar Código
Modificar `lib/products.ts`:

```typescript
{
  id: 'juguete-interactivo-perro',
  name: 'Juguete Interactivo Inteligente',
  // Cambiar esto:
  // image: 'https://images.unsplash.com/...'
  // Por esto:
  image: '/products/CJDOG001/main.jpg',
  images: [
    '/products/CJDOG001/gallery/01.jpg',
    '/products/CJDOG001/gallery/02.jpg',
    '/products/CJDOG001/gallery/03.jpg',
  ],
  // ... resto de propiedades
}
```
